# Мой учебный репозиторий

Это репозиторий для хранения моих работ (домашних заданий, практик и проектов) по курсу Git.

## Содержание
- [Домашние задания](./course-1/homeworks/)
- [Практики](./course-1/practices/)
- [Проекты](./course-1/projects/)

# Использование
1. Открыть папку нужного задания
2. внутри папки с нужным названием работы смотреть файлы "task.md" (само задание), "solution/" (код выполненой работы), README.md (инструкция по запуску)
3. если папка с работой не содержит папку "solution", считать это за отсутствие кода
