# Практика
## Уровень 1
### Задание 1: Проверка числа
1. Создайте новый проект C# (консольное приложение).
2. Объявите переменную int number = 10;
3. Напишите программу, которая выводит:
"Число положительное", если number > 0;
"Число отрицательное", если number < 0;
"Число равно нулю", если number == 0.

### Задание 2: Проверка возраста
1. Считайте с консоли, введенный возраст и сохраните в переменную age.
2. Используя if-else, выведите:
"Вы совершеннолетний", если возраст больше или равен 18;
"Вы несовершеннолетний", если меньше 18.

### Задание 3: Проверка чётности числа
1. Объявите переменную int num = 7;.
2. Используйте условие и оператор %, чтобы проверить:
если число делится на 2 без остатка, то вывести "Число чётное";
иначе вывести "Число нечётное".

## Уровень 2

### Задание 4. Логические операторы
1. Объявите две переменные: int a = 5, b = -2;
2. Составьте условия:
Если оба числа положительные, то вывести "Оба числа положительные".
Если хотя бы одно число положительное, то вывести "Хотя бы одно число положительное".
Если число a не положительное, то вывести "a не положительное".

### Задание 5. Калькулятор оценок 
1. Объявите переменную int grade.
2. Считайте оценку ученика с клавиатуры (Console.ReadLine()).
3. Используйте условия:
Если оценка меньше 3, то вывести "Неудовлетворительно".
Если оценка 3, то вывести "Удовлетворительно".
Если оценка 4, то вывести "Хорошо".
Если оценка 5, то вывести "Отлично".

### Задание 6. Калькулятор
1. Получите от пользователя 2 числа и операцию (+,-,*,/)
2. Выполните действие относительно операции, которую запросил пользователь

### Задание 7. Проверка пароля
1. Создайте переменную password и запишите туда любой пароль
2. Получите пароль от пользователя
3. Если пароль совпадает то выводим "Доступ получен"
4. Если не совпадают, то "Доступ запрещен".
